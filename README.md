# MCOtterShades
A Minecraft shader pack built for modern rendering standards, focusing on high visual fidelity and performance optimization.

## Quick Start
1. Download a compatible Minecraft version with OptiFine or Iris.
2. Clone this repository into your `.minecraft/shaderpacks` folder.
3. Enable "MCOtterShades" in the shader settings.
